# Q1 Printer Monitor

This service monitors a Qidi Q1 Pro 3D printer and sends a notification when the printer's state changes from "printing" to any other state.

Additional process using a model for 3d printing issue detections (Javiai/3dprintfails-yolo5vs) is running wheh the printer is printing.
